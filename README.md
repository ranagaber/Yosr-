# YOSR (يُسر) A Multimodal Intelligent Tutoring System for Egyptian Students
YOSR is a multimodal intelligent tutoring system that transforms lecture slides into accessible educational content in Egyptian Arabic. The system leverages Vision-Language Models (VLMs), Large Language Models (LLMs), and speech technologies to generate explanations, study guides, quizzes, and audio summaries from educational material.

This repository contains the training code, data preprocessing pipelines, evaluation scripts, and utilities used to develop the VLM model behind YOSR.
